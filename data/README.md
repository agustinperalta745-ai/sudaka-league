# Data de Copa Interdivisional

Este es el archivo fuente que consume la UI para Interdivisional SLP T25: `data/copaInterdivisionalActiva.json`.

- `data/copaInterdivisionalActiva.json`: temporada activa mostrada en la card "Copa Interdivisional SLP".
- `data/interdivisional.json`: historial de temporadas anteriores (no es la fuente activa T25).
